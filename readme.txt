Golang, known for its simplicity and efficiency, is a statically typed, compiled language created by Google.

Your interpreter, designed in Java, can only interpret simple algorithms written in a minimal subset of Golang. It supports the following 10 algorithms:

Sum of First N Numbers
Factorial of N
GCD of Two Numbers
Reverse a Number
Check if a Number is Prime
Check if a Number is Palindrome
Find the Largest Digit in a Number
Sum of Digits
Multiplication Table
Nth Fibonacci Number
